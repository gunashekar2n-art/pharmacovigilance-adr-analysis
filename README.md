# Pharmacovigilance ADR Analysis

This project analyzes adverse drug reactions (ADR) using SQL.

## Dataset
A pharmacovigilance dataset containing drug reactions, patient demographics, and outcomes.

## Key Analyses
- Top drugs causing ADR
- Fatal drug reactions
- Serious vs non-serious ADR distribution
- ADR trends over time

## Tools Used
- SQL
- Data Cleaning
- Data Analysis
- GitHub
